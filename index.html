How to Apply CSS to HTML

To apply CSS to HTML, you can use the following methods:

  1. Inline CSS: Here's an example of changing text to red using inline CSS. The code below is enclosed in <code> for inline code and <pre> for block code: 
    <p style="color: red;">This text will be red.</p>

    This text will be red.
